# YouCut - 专业视频编辑应用

> 本站**对外展示**以 `showcase/pro7.html` 为准（项目介绍 → Demo 双列卡片 → 技术介绍 → 八大模块）。本文件为自 GitHub 同步的备忘，可与详情页交叉对照。

YouCut 是一款功能强大的视频编辑应用，提供丰富的视频处理功能，包括AI智能编辑、特效处理、滤镜美化、动画效果等。

## 🎬 功能展示

### 基础功能
<img src="https://raw.githubusercontent.com/cgeffect/YouCut/1.1.1/preview/%E5%9F%BA%E7%A1%80.PNG" width="200" alt="基础功能">

**截图对应能力：** 多轨时间线浏览与缩放；素材导入、拖拽入轨与剪切拼接；音轨与画轨对齐；片段裁剪与基础转场预览；导出前成片预览入口。

### 画布功能
<img src="https://raw.githubusercontent.com/cgeffect/YouCut/1.1.1/preview/%E7%94%BB%E5%B8%83.gif" width="200" alt="画布功能">

**截图对应能力：** 竖屏 / 横屏等画幅预设；纯色、渐变与图片背景；背景模糊与主体分离感；元素对齐与吸附；安全区与标题安全线辅助。

### 动画效果

### 滤镜效果

## 🚀 主要功能

YouCut 提供 8 个主要功能模块，每个模块都包含丰富的子功能（与下表一致，可在详情页 `showcase/pro7.html` 同步查看）：

### 1. 画布 (Canvas)
- 多种画布尺寸选择
- 背景模糊效果
- 渐变背景
- 图片背景
- 对齐方式调整

### 2. AI智能 (Brain)
- 智能人像分割
- AI背景替换
- 智能抠图
- 自动场景识别

### 3. 特效 (Effect)
- 转场特效
- 视觉特效
- 粒子效果
- 动态效果

### 4. 滤镜 (Filter)
- 多种滤镜风格
- 实时预览
- 强度调节
- 自定义滤镜

### 5. 美颜 (Beauty)
- 肤色调整
- 磨皮美颜
- 瘦脸瘦身
- 眼部增强

### 6. 动画 (Animation)
- 入场动画
- 出场动画
- 转场动画
- 关键帧动画

### 7. 裁剪 (Crop)
- 自由裁剪
- 比例裁剪
- 旋转调整
- 翻转功能

### 8. 旋转 (Rotate)
- 90度旋转
- 180度旋转
- 270度旋转
- 自由角度旋转

> 📖 **详细功能文档**: 仓库内 [FEATURES.md](https://github.com/cgeffect/YouCut/blob/1.1.1/FEATURES.md) 含完整子功能说明（外链至 GitHub）。

## 🛠 技术栈

- **开发语言**: Swift
- **UI框架**: UIKit + SnapKit
- **视频处理**: CGMetal + MetalPetal
- **AI处理**: Core ML + Vision
- **依赖管理**: CocoaPods

## 📱 系统要求

- iOS 15.0+
- Xcode 15.0+
- Swift 5.0+
