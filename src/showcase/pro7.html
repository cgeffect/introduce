pro7 · iOS · 剪辑与包装

专业视频编辑应用

参考实现基于公开分支 1.1.1，侧重画布、AI 人像、特效与滤镜等移动端剪辑能力展示。

一、项目介绍

这是一款面向 iOS 的专业向视频编辑应用：在手机上完成多轨道思路下的包装与出片，覆盖画布比例与背景、AI 人像分割与换背景、特效与粒子、滤镜风格、美颜美体、入场出场与转场动画，以及裁剪与旋转等常见剪辑动作，适合营销短视频、口播包装与轻量创作流程。

产品形态上强调「模块清晰、预览跟手」：各能力分区明确，关键步骤可实时预览，减少反复导出试错的成本。

二、效果与 Demo 演示

资源说明：下列截图与 GIF 与仓库 README 中的功能展示一致，文件托管在 GitHub（raw.githubusercontent.com）。若图片长时间空白，请检查网络能否访问 GitHub，必要时需科学上网；GIF 体积较大，首次加载可能较慢。每个卡片内为该预览所展示的具体能力说明。

基础功能

  • 多轨时间线浏览与缩放，片段剪切、拼接与排序
  • 素材导入、拖拽入轨，音轨与画轨对齐
  • 基础转场预览，导出前成片预览入口

画布功能

  • 竖屏 / 横屏等画幅预设，安全区与标题安全线
  • 纯色、渐变与图片背景，背景模糊突出主体
  • 元素对齐、吸附与分布辅助

动画效果

  • 入场 / 出场动画预设，片段间转场动画
  • 关键帧控制位移、缩放、透明度
  • 动效时长与口播节奏对齐

滤镜效果

  • 多风格 LUT / 滤镜预设，实时预览
  • 强度滑杆与肤色、对比度微调
  • 自定义参数保存与复用

人像处理

  • 人像分割与边缘羽化，一键换背景（色块 / 图片）
  • 抠像后主体清晰度与色彩增强
  • 与美颜衔接：磨皮、瘦脸等参数调节

特效处理

  • 粒子、光效类画面包装
  • 贴纸、花字与强调框、箭头等引导元素
  • 特效与时间点绑定，控制起止与强度

三、技术介绍

客户端采用 Swift 与 UIKit，布局使用 SnapKit；视频与图像管线侧结合 MetalPetal 等方案做 GPU 侧滤镜与合成；AI 相关能力使用 Core ML 与 Vision 做人像分割、场景理解等；依赖通过 CocoaPods 管理。

运行与开发环境方面，仓库说明为 iOS 15.0+、Xcode 15.0+、Swift 5.0+；更细的模块划分与接口级说明以仓库内文档为准。

更细的模块与接口说明见仓库内 FEATURES.md，与上文 Demo 预览及下文八大模块为同一套能力描述。

四、八大功能模块

与产品内模块划分一致，下列为各模块对外可见的核心子能力（与仓库 README 列表对齐）。

1. 画布 (Canvas)

  • 多种画布尺寸选择
  • 背景模糊效果
  • 渐变背景、图片背景
  • 对齐方式调整

2. AI 智能 (Brain)

  • 智能人像分割
  • AI 背景替换
  • 智能抠图
  • 自动场景识别

3. 特效 (Effect)

  • 转场特效、视觉特效
  • 粒子效果、动态效果

4. 滤镜 (Filter)

  • 多种滤镜风格、实时预览
  • 强度调节、自定义滤镜

5. 美颜 (Beauty)

  • 肤色调整、磨皮美颜
  • 瘦脸瘦身、眼部增强

6. 动画 (Animation)

  • 入场动画、出场动画、转场动画
  • 关键帧动画

7. 裁剪 (Crop)

  • 自由裁剪、比例裁剪
  • 旋转调整、翻转功能

8. 旋转 (Rotate)

  • 90° / 180° / 270° 旋转
  • 自由角度旋转
