pro5 · macOS · Skia 图文渲染

skia-macos 图文渲染系统

Skia 图形栈在 macOS 上做模板渲染与高清导出。

项目介绍

基于 Skia 在 macOS 上搭建的图文渲染能力，用于模板化制图、批量出图与与设计规范一致的平面物料生产；可与业务系统对接，由结构化数据驱动版式与导出。

相关素材与说明可置于 resource/pro5/。

技术介绍

渲染侧以 Skia 为核心做矢量与位图合成，字体子集化、图片解码与色彩空间在导出前一次性烘焙；宿主侧常用 Swift/Objective-C++ 桥接，版式数据可由 JSON 或业务中台下发。
